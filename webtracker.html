For now, simply rename the file to [yourname].zip, and extract to enjoy.

Import Samples

You can import samples from .mod files or from any file your browser can decode. Most browsers can decode .wav files. Other choices are: ogg, opus, WebM/WebA, and MP3. If your file doesn't open in this browser, save your work and open it in another, then import the file.
